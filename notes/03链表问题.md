非递归方式：
```java
public ListNode reverseList1(ListNode head) {
    ListNode pre=null;
    ListNode cur=head;

    while(cur!=null){
        ListNode next=cur.next;
        cur.next=pre;
        pre=cur;
        cur=next;
    }
    return pre;
}
```
递归方式：
```java
public ListNode reverseList(ListNode head) {
    if(head==null || head.next==null){
        return head;
    }

    ListNode next=head.next;
    head.next=null;
    ListNode newNode=reverseList(next);
    next.next=head;
    return newNode;
}
```
